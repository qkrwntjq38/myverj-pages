노션 대안 요약

요청: 오빠가 스마트폰에서 보기 좋게 정리한 요약 페이지입니다.

추천 도구(요약)

  • Obsidian — 로컬 마크다운 기반, 오프라인 우선, 개인 지식관리에 최적.
  • Logseq — 블록/그래프 중심, 연구·노트 연결에 강함(로컬 사용 가능).
  • OneNote — 필기·이미지 중심, Office 연동이 필요할 때 좋음.
  • Airtable / Coda — 데이터베이스+문서 결합, 팀·프로세스 관리에 추천.
  • Joplin — 오픈소스 마크다운 노트, 암호화·로컬 우선.

GitHub Pages로 배포 방법(요약)

  1. 내가 준비한 파일(zip)를 업로드 → GitHub에 새 레포 생성
  2. 레포 Settings → Pages 활성화(브랜치: main, root)
  3. 도메인(myverj.my) DNS에 CNAME 또는 A 레코드 추가
  4. HTTPS 자동 적용 후 접속 확인
